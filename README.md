# 🚀 TaskFlow - Gamified To-Do List Web Application

TaskFlow is a modern and interactive To-Do List web application designed to boost productivity through gamification. Users can create, manage, and track tasks while earning XP, leveling up, and maintaining streaks.

## 🌟 Features

* ✅ Create, complete, and delete tasks
* 🎯 Priority levels (High, Medium, Low)
* 📂 Task categorization
* ⏰ Due date and time management
* ⚡ XP reward system
* 🏆 Level progression system
* 🔥 Daily streak tracking
* 📊 Progress analytics dashboard
* 🔍 Task filtering and sorting
* 🎨 Modern responsive UI
* ✨ Smooth animations and effects

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/vaseemsadikkali/todolist.git
```

2. Open the project folder

```bash
cd todolist
```

3. Launch the application

Open `todolist.html` in your browser.
